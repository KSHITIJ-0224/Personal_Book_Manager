A full-stack Personal Book Manager built using the MERN stack with Next.js for the frontend.

This application allows users to securely manage their personal reading collection — track books, monitor reading status, and maintain an organized digital bookshelf.

🚀 Features
🔐 Authentication
User Signup
User Login
JWT-based authentication
Protected routes
Secure password hashing

📚 Book Management

Users can:
Add a new book (title, author, tags, status)
Edit existing books
Delete books
Filter books by:

📖 Want to Read
📘 Reading
✅ Completed

Tags
📊 Dashboard
Total number of books
Status-wise book count
Clean and minimal UI
Quick status updates

🛠 Tech Stack
Frontend
Next.js (App Router)
Tailwind CSS
Axios (API communication)
Backend
Node.js
Express.js
MongoDB + Mongoose
JWT Authentication
Bcrypt for password hashing

Database
MongoDB Atlas
